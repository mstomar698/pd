### To run locally:

```powershell
# Clone the repo
git clone https://github.com/mstomar698/pd.git
# Go to the directory
cd pd
# Install dependencies
npm install
# Run the tool
npm run dev
# Build the tool
npm run build
```

### To use the tool:

```powershell
# Install the tool globally
npm install -g pd
# Run the tool
pd
# for help
pd --help # or pd -h
# Uninstall the tool
npm uninstall -g pd
```
